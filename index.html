What's inside

Every module you need, wired together

⬡

Row-based multi-tenancy

TenantContext with ThreadLocal resolution. Every query is tenant-scoped automatically. X-Tenant-ID header resolved before your code runs.

◈

JWT + refresh token rotation

15-minute access tokens, 7-day refresh tokens stored in DB. Rotation on every use, revocation on logout. All configurable via env vars.

RBAC with method security

USER / MANAGER / ADMIN roles with granular permissions. @PreAuthorize on any method. Add new roles in a single enum.
